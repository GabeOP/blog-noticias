# Descrição
Projeto fullstack de um blog com **autorização** e **autenticação** de usuário. Usando um perfil de ADM, é possível
criar postagens para todos os usuários cadastrados.

# Tecnologias
- Java, Javascript
- HTML, CSS
- Spring boot, Spring security
- Docker
- PostgreSQL
